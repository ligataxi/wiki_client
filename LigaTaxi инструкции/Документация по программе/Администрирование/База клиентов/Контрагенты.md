В данном разделе хранятся данные о контрагентах - организациях, которые пользуются услугами службы такси по безналичному расчету. Для них можно создать отдельные типы заказов, которые автоматически подставляются в карточке нового заказа, стоит оператору только выбрать нужную фирму. К типу заказа можно привязать отдельный тариф, который будет автоматически применяться для таких поездок. 

Для каждой фирмы, руководство такси может указать ограничение по балансу. Если у контрагента будет недостаток суммы на балансе  для оплаты поездки, программа оповестит об этом оператора, и запретит создавать такой заказ.   

Выставление счетов контрагентам происходит в течении нескольких минут. Для этого руководству такси необходимо зайти в Отчеты - Заказы, выбрать в фильтре нужного Контрагента, указать дату и нажать кнопку Сформировать отчет. А также с  помощью нашей уникальной разработки Редактор отчета, есть возможность добавить в отчет всю информацию о заказе, которая может потребоваться контрагентам для сверки. 

Если фирмам партнерам требуется загрузить данные в 1С, то достаточно скачать отчет в формате Excel и загрузить его в программу 1С. Кроме этого, для каждого контрагента доступен личный кабинет, в котором отображается информация по всем поездкам данной организации.

Для добавления контрагента необходимо нажать на кнопку “Добавить”, и заполнить следующие поля настроек:

* Название - укажите произвольное название для данного контрагента
* Полное имя - укажите полное название юридического или имя физического лица - контрагента
* Кодовая фраза - укажите кодовую фразу, при указании которой оператор сможет определить, что поездку будет совершать контрагент
* Адрес электронной почты - укажите актуальный адрес электронной почты контрагента
* Типы заказов - укажите типы заказов, с которыми будут создаваться заявки от данного контрагента
* Минимальный баланс - укажите значение минимального баланса. Если текущий баланс контрагента опустится ниже минимального, контрагент не сможет совершать поездки. Можно указывать отрицательное значение
* Номера телефонов укажите список номеров, с которых разрешено создавать заказы для данного Контрагента. Введите номер телефона и нажмите клавишу Enter. Оставьте поле пустым для разрешения создания заказов с любых номеров
* Адрес - укажите адрес контрагента
* Реквизиты - укажите необходимые реквизиты контрагента
* Комментарий - при необходимости в данном поле можно оставить комментарий, который будут видеть только пользователи с соответствующими правами
* Отчеты - укажите отчеты, к которым сможет иметь доступ контрагент (работает только на просмотр). Желательно указывать только настроенные конкретно под данного контрагента отчеты

После внесения всех настроек необходимо нажать на кнопку “Сохранить”.
